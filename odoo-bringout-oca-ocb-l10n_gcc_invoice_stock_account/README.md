# Gulf Cooperation Council WMS Accounting


Adds Arabic as a secondary language for the lots and serial numbers
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_gcc_invoice_stock_account
```

## Dependencies

- l10n_gcc_invoice
- stock_account
