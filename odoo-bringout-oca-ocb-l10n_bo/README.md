# Bolivia - Accounting


Bolivian accounting chart and tax localization.

Plan contable boliviano e impuestos de acuerdo a disposiciones vigentes

    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_bo
```

## Dependencies

This addon depends on:
- l10n_multilang
