# Chile - Accounting


Chilean accounting chart and tax localization.
Plan contable chileno e impuestos de acuerdo a disposiciones vigentes.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_cl
```

## Dependencies

- contacts
- base_vat
- l10n_latam_base
- l10n_latam_invoice_document
- uom
- account
